About

This is a webscraper that returns a table containing information about research articles relating to the given search parameters.

This application use jquery to send a post request to a flask application which is hosted on google cloud.
The flask application runs a python script that gathers articles from google scholar which are displayed in the table.

To use the scraper in the title of your project and a few keywords which are related to your project.

An example would be :

Title: Life cycle assessment of hydrogen production from biomass gasification
Keywords: hydrogen syngas assessment emission gasification energy
